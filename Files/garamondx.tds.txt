This is an expertized version of URW GaramondNo8 with a complete set of LaTeX support files plus a matching math option. There is no need to install the ugm package as well. Version 1.01 added improved metrics and kerns for parentheses, brackets and braces, while version 1.06 provides enhanced support for textcomp glyphs. Version 1.07 corrects some small cap glyph errors, 1.08 adds options osf, osfI and sups, while 1.09 adds more accented glyphs covering almost all of T1 encoding. This version, 1.099, adds an upright mu for siunitx users.
